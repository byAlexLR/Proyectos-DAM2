# Gestor de Productos e Inventario (API & MySQL)

Aplicación de escritorio desarrollada en Python para la gestión de inventario y productos. Permite visualizar productos, filtrar búsquedas y exportar informes en formatos PDF y Excel.

## Instalación y Configuración

### 1. Clonar o descargar el repositorio

Descarga los archivos del proyecto en el ordenador.

### 2. Instalar dependencias

Abre el terminal (o consola de comandos) en la carpeta del proyecto y ejecuta el siguiente comando para instalar las librerías necesarias:

```bash
pip install requests mysql-connector-python reportlab fpdf pandas openpyxl
```
